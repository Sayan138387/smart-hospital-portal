🏥 Smart Hospital Portal

A simple and interactive Smart Hospital Web Application built using HTML, CSS, and JavaScript.
This project focuses on improving user experience for hospital services with modern UI and useful features.

✨ Features

Doctor Search & Filter

Appointment & Contact Pages

Medicine Store with Cart System

Health Tracker

Dark Mode Support

Responsive Design

🛠 Tech Stack

HTML

CSS

JavaScript

🚀 Purpose

This project is created to practice frontend development and build a real-world healthcare themed web application.

📌 Status
Project is under continuous improvement. More features like AI Symptom Checker and Live Chat will be added soon.
